📚 Readify - Library Management System
## Readify is a modern and intuitive library management system designed to make book discovery and management a breeze! Built with HTML, CSS, JavaScript, PHP, and MySQL, this platform offers a clean and user-friendly experience for both users and admins.

# ✨ Features
- ## 👤 User Features:
  - 🔐 Secure Authentication: Users can create accounts and log in securely.

- ## 🔍 Book Discovery:
  -  Browse books by genre, including Fantasy, Romance, Sci-Fi, Horror, and more.

- ## 📊 Personalized Dashboard:
  -  Users can view available books and their details.

- ## 👨‍💻 Admin Features:
  - 📖 Add New Books: Admins can add books to the system, including details like title, author, year of publication, and description.

  - 🔒 Role-Based Access: Admins have exclusive access to the admin panel, ensuring secure management of the library.

- ## 🌟 General Features:
  - 🎨 Figma-Designed UI: The platform’s UI/UX was pre-designed using Figma, incorporating ready-made libraries for a modern and sleek interface.

- ## 🌐 Online MySQL Database:
  -  The system is connected to an online MySQL database for efficient data storage and retrieval.

- ## 🛠️ Technologies Used
  - Frontend: HTML, CSS, JavaScript
  - Backend: PHP
  - Database: MySQL (hosted online)
  - Design: Figma
